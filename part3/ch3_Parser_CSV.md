csv data table like this...
```text
1   2000/10/05  02  08  10  13  27  30
2   2000/10/12  01  09  16  20  21  43
3   2000/10/19  01  05  15  31  36  38
4   2000/10/26  16  18  26  27  34  40
5   2000/11/02  09  15  21  23  27  28
```

code is here
```fsharp
#r @"./packages/FSharp.Data/lib/net40/FSharp.Data.dll"
open FSharp.Data
open System

module Test =

    let str = "1,2000/10/05,02,08,10,13,27,30
               2,2000/10/12,01,09,16,20,21,43
               3,2000/10/19,01,05,15,31,36,38
               4,2000/10/26,16,18,26,27,34,40
               5,2000/11/02,09,15,21,23,27,28"

    // ヘッダーがない場合は、 hasHeaders = false としておく
    let CSVdata = CsvFile.Parse ( str, hasHeaders = false )

    // スピードアップのためにキャッシュしておく
    CSVdata.Cache ()
```
.  
column1 だけとりだす
```fsharp
[ for row in CSVdata.Rows ->  row.[1] ]
|> Seq.iter ( fun arr -> printfn "%A" arr )
```
result
```
"2000/10/05"
"2000/10/12"
"2000/10/19"
"2000/10/26"
"2000/11/02"
```
.  
column0 にて逆順にソートする
```fsharp
[for row in CSVdata.Rows -> row ]
|> Seq.sortBy ( fun arr ->  - Int32.Parse arr.[0] )
|> Seq.iter ( fun arr -> printfn "%A" arr )
```
result
```
[|"5"; "2000/11/02"; "09"; "15"; "21"; "23"; "27"; "28"|]
[|"4"; "2000/10/26"; "16"; "18"; "26"; "27"; "34"; "40"|]
[|"3"; "2000/10/19"; "01"; "05"; "15"; "31"; "36"; "38"|]
[|"2"; "2000/10/12"; "01"; "09"; "16"; "20"; "21"; "43"|]
[|"1"; "2000/10/05"; "02"; "08"; "10"; "13"; "27"; "30"|]
```
